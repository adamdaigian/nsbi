title: GitHub Pages
description: Deploy Polaris dashboards to GitHub Pages
---

## Setup

### 1. Configure Base Path

If deploying to `https://username.github.io/repo-name/`, set the base path in `polaris.config.ts`:

```ts
export default {
  basePath: "/repo-name/",
};
```

### 2. Build

```bash
npx polaris build --project .
```

### 3. GitHub Actions Workflow

Create `.github/workflows/deploy.yml`:

```yaml
name: Deploy to GitHub Pages

on:
  push:
    branches: [main]

permissions:
  contents: read
  pages: write
  id-token: write

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: 20
      - run: npm install
      - run: npx polaris build --project .
      - uses: actions/upload-pages-artifact@v3
        with:
          path: dist

  deploy:
    needs: build
    runs-on: ubuntu-latest
    environment:
      name: github-pages
      url: ${{ steps.deployment.outputs.page_url }}
    steps:
      - uses: actions/deploy-pages@v4
        id: deployment
```

### 4. Enable GitHub Pages

Go to **Settings → Pages → Source** and select **GitHub Actions**.
