## Setup

### 1. Build Settings

In **Site Settings → Build & Deploy**:

| Setting | Value |
|---------|-------|
| Build Command | `npx polaris build --project .` |
| Publish Directory | `dist` |

### 2. SPA Routing

Polaris generates a `_redirects` file in `dist/` during build:

```
/*    /index.html   200
```

This handles client-side routing automatically.

### 3. Deploy

Connect your Git repository to Netlify, or deploy manually:

```bash
npx polaris build --project .
npx netlify deploy --dir=dist --prod
```
