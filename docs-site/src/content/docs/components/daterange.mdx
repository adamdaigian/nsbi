---
title: DateRange
description: A date range picker with preset ranges and a custom calendar selector.
---

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| name | string | Yes | — | Filter variable name. Use `${name}` in SQL to reference the selected range. The value is an object with `from` and `to` date fields. |
| label | string | No | — | Label displayed above the date range picker. |

The component includes built-in presets: Last 7 days, Last 30 days, This month, This quarter, This year, and a Custom mode with a dual-month calendar.

## Example

```mdx
<DateRange name="date_range" label="Date Range" />

```sql
-- name: filtered_orders
SELECT
  DATE_TRUNC('day', order_date) AS day,
  COUNT(*) AS order_count,
  SUM(total) AS revenue
FROM orders
WHERE order_date >= '${date_range.from}'
  AND order_date <= '${date_range.to}'
GROUP BY 1
ORDER BY 1
```

<LineChart
  data="filtered_orders"
  x="day"
  y="revenue"
  xFmt="MMM d"
  yFmt="$#,##0"
  title="Revenue Over Time"
/>
```

### Combined with other filters

```mdx
<Grid cols={3}>
  <DateRange name="period" label="Period" />
  <Dropdown name="channel" options={["online", "retail", "wholesale"]} label="Channel" />
  <Dropdown name="region" options={["North", "South", "East", "West"]} label="Region" />
</Grid>

```sql
-- name: sales_filtered
SELECT product, SUM(amount) AS total
FROM sales
WHERE sale_date >= '${period.from}'
  AND sale_date <= '${period.to}'
  AND channel = '${channel}'
  AND region = '${region}'
GROUP BY product
ORDER BY total DESC
```

<BarChart data="sales_filtered" x="product" y="total" title="Sales by Product" />
```
