---
title: LineChart
description: A line chart for visualizing trends and time-series data.
---

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| data | string | Yes | — | Name of the SQL query to use as the data source. |
| x | string | Yes | — | Column to map to the x-axis. |
| y | string | Yes | — | Column to map to the y-axis (numeric values). |
| color | string | No | — | Column used to split data into multiple colored lines. |
| title | string | No | — | Title displayed above the chart. |
| yFormat | string | No | — | Vega-Lite format string for y-axis labels (e.g. `"$~s"` for compact currency, `".0%"` for percentages). |
| xTimeUnit | string | No | — | Vega-Lite time unit for x-axis (e.g. `"yearmonth"`, `"yearweek"`, `"yearmonthdate"`). |

## Example

```mdx
```sql
-- name: monthly_revenue
SELECT
  date_trunc('month', order_date) AS month,
  product_category,
  SUM(revenue) AS total_revenue
FROM orders
GROUP BY month, product_category
ORDER BY month
```

<LineChart
  data="monthly_revenue"
  x="month"
  y="total_revenue"
  color="product_category"
  xTimeUnit="yearmonth"
  yFormat="$~s"
  title="Monthly Revenue by Category"
/>
```
