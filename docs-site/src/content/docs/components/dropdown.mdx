---
title: Dropdown
description: A single-select dropdown filter that sets a template variable for SQL queries.
---

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| name | string | Yes | — | Filter variable name. Use `${name}` in SQL to reference the selected value. |
| options | `{ label: string; value: string }[]` | No | — | Static list of options. Each object has a `label` (displayed) and `value` (used in SQL). |
| label | string | No | — | Label displayed above the dropdown. |
| defaultValue | string | No | — | Initial selected value. Seeds the filter context on mount. |

## Example

````mdx
<Dropdown
  name="plan"
  options={[
    { label: "Free", value: "free" },
    { label: "Starter", value: "starter" },
    { label: "Pro", value: "pro" },
    { label: "Enterprise", value: "enterprise" }
  ]}
  defaultValue="free"
  label="Plan"
/>

```sql
-- name: users_by_plan
SELECT * FROM users WHERE plan = '${plan}'
```

<DataTable data="users_by_plan" />
````
