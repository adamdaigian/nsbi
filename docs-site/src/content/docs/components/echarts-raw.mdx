---
title: ECharts (Raw)
description: A low-level escape hatch for rendering any Apache ECharts configuration directly.
---

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| options | EChartsOption | Yes | — | A full Apache ECharts option object. See the [ECharts documentation](https://echarts.apache.org/en/option.html) for all available fields. |
| title | string | No | — | Title displayed above the chart (rendered outside ECharts). |
| subtitle | string | No | — | Subtitle displayed below the title (rendered outside ECharts). |
| height | number | No | `400` | Height of the chart in pixels. |
| notMerge | boolean | No | `false` | When `true`, the new options replace existing options entirely instead of merging. Useful when dynamically switching chart types. |

## Example

```mdx
```sql
-- name: funnel_data
SELECT stage, users
FROM (
  VALUES
    ('Visited Site', 12000),
    ('Signed Up', 5200),
    ('Activated', 3100),
    ('Subscribed', 1400),
    ('Renewed', 950)
) AS t(stage, users)
```

<ECharts
  options={{
    tooltip: { trigger: "item" },
    series: [
      {
        type: "funnel",
        left: "10%",
        width: "80%",
        label: { position: "inside" },
        data: [
          { value: 12000, name: "Visited Site" },
          { value: 5200, name: "Signed Up" },
          { value: 3100, name: "Activated" },
          { value: 1400, name: "Subscribed" },
          { value: 950, name: "Renewed" },
        ],
      },
    ],
  }}
  title="Conversion Funnel"
  subtitle="End-to-end user journey"
  height={500}
/>
```
