---
title: ScatterPlot
description: A scatter plot for exploring relationships between two numeric variables.
---

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| data | string | Yes | — | Name of the SQL query to use as the data source. |
| x | string | Yes | — | Column to map to the x-axis (numeric). |
| y | string | Yes | — | Column to map to the y-axis (numeric). |
| series | string | No | — | Column used to color-code points into distinct groups. |
| size | string | No | — | Column used to scale the size of each point (bubble chart). |
| title | string | No | — | Title displayed above the chart. |
| subtitle | string | No | — | Subtitle displayed below the title. |
| height | number | No | `400` | Height of the chart in pixels. |
| colorPalette | string[] | No | — | Custom array of color hex codes for the series. |

## Example

```mdx
```sql
-- name: customer_metrics
SELECT
  customer_id,
  total_lifetime_spend,
  avg_order_value,
  order_count,
  segment
FROM customer_summary
WHERE total_lifetime_spend > 0
```

<ScatterPlot
  data="customer_metrics"
  x="total_lifetime_spend"
  y="avg_order_value"
  size="order_count"
  series="segment"
  title="Customer Spend vs. Order Value"
  subtitle="Point size represents number of orders"
/>
```
