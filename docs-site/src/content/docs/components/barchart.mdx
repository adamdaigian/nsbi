---
title: BarChart
description: A bar chart for comparing values across categories.
---

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| data | string | Yes | — | Name of the SQL query to use as the data source. |
| x | string | Yes | — | Column to map to the category axis. |
| y | string | Yes | — | Column to map to the value axis (numeric values). |
| color | string | No | — | Column used to split data into grouped or stacked bar series. |
| stack | boolean | No | `false` | Whether to stack bars on top of each other instead of grouping side by side. |
| title | string | No | — | Title displayed above the chart. |
| yFormat | string | No | — | Vega-Lite format string for value-axis labels (e.g. `"$~s"` for compact currency). |
| xTimeUnit | string | No | — | Vega-Lite time unit for x-axis (e.g. `"yearmonth"`). Use when x is a date field. |

## Example

```mdx
```sql
-- name: sales_by_region
SELECT region, sales_channel, SUM(amount) AS total_sales
FROM sales
WHERE sale_date >= '2025-01-01'
GROUP BY region, sales_channel
ORDER BY total_sales DESC
```

<BarChart
  data="sales_by_region"
  x="region"
  y="total_sales"
  color="sales_channel"
  stack
  yFormat="$~s"
  title="Sales by Region"
/>
```

### Grouped bars

Omit `stack` to show bars side by side:

```mdx
<BarChart data="sales_by_region" x="region" y="total_sales" color="sales_channel" title="Sales by Region" />
```
