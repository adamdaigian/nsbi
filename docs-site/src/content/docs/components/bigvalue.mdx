---
title: KPI
description: A KPI card that highlights a single important metric with optional comparison delta.
---

In MDX pages, use `<KPI>` to display key metrics. It resolves query names automatically and renders as a prominent value card.

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| data | string | Yes | — | Name of the SQL query to use as the data source (uses the first row). |
| value | string | Yes | — | Column containing the primary metric value to display. |
| title | string | No | — | Label displayed above the metric value. |
| format | string | No | — | Format string for the primary value (e.g. `"usd_compact"`, `"num0"`, `".0%"`). |
| comparison | string | No | — | Column containing a comparison value (e.g. delta from previous period). |
| comparisonFormat | string | No | — | Format string for the comparison value. |
| comparisonLabel | string | No | — | Label for the comparison indicator (e.g. `"vs last month"`). |
| isUpGood | boolean | No | `true` | Whether an upward trend is positive (green) or negative (red). Set to `false` for metrics like churn. |

## Example

````mdx
```sql
-- name: revenue_kpi
SELECT
  SUM(revenue) AS current_revenue,
  SUM(revenue) - LAG(SUM(revenue)) OVER (ORDER BY period) AS revenue_delta
FROM monthly_financials
GROUP BY period
ORDER BY period DESC
LIMIT 1
```

<Grid cols={3}>
  <KPI
    data="revenue_kpi"
    value="current_revenue"
    title="Total Revenue"
    format="usd_compact"
    comparison="revenue_delta"
    comparisonFormat="usd_compact"
    comparisonLabel="vs last month"
  />
</Grid>
````
