---
title: Sparkline
description: A compact inline chart for showing trends in a small space.
---

:::note
Sparkline is available as a library component but is not directly usable in MDX dashboard pages. It accepts raw data arrays rather than query name strings, so it cannot be wired up to SQL queries like other dashboard components.
:::

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| data | string | Yes | — | Name of the SQL query to use as the data source. |
| y | string | Yes | — | Column containing the numeric values to plot. |
| height | number | No | `32` | Height of the sparkline in pixels. |
| width | number | No | `120` | Width of the sparkline in pixels. |
| color | string | No | — | Color of the sparkline stroke (e.g. `"#4f46e5"` or `"steelblue"`). |

## Example

````mdx
```sql
-- name: daily_active_users
SELECT
  activity_date,
  COUNT(DISTINCT user_id) AS dau
FROM user_activity
WHERE activity_date >= CURRENT_DATE - INTERVAL '30 days'
GROUP BY activity_date
ORDER BY activity_date
```

<Sparkline
  data="daily_active_users"
  y="dau"
  height={28}
  width={140}
  color="#4f46e5"
/>
````
