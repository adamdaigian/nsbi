---
title: AreaChart
description: A stacked area chart for visualizing volume and composition over time.
---

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| data | string | Yes | — | Name of the SQL query to use as the data source. |
| x | string | Yes | — | Column to map to the x-axis (typically a date). |
| y | string | Yes | — | Column to map to the y-axis (numeric values). |
| color | string | No | — | Column used to split data into stacked area series. |
| title | string | No | — | Title displayed above the chart. |
| yFormat | string | No | — | Vega-Lite format string for y-axis labels (e.g. `"$~s"`, `"#,##0"`). |
| xTimeUnit | string | No | — | Vega-Lite time unit for x-axis (e.g. `"yearmonth"`, `"yearweek"`). |

## Example

```mdx
```sql
-- name: weekly_signups
SELECT
  date_trunc('week', created_at) AS week,
  signup_source,
  COUNT(*) AS signups
FROM users
GROUP BY week, signup_source
ORDER BY week
```

<AreaChart
  data="weekly_signups"
  x="week"
  y="signups"
  color="signup_source"
  xTimeUnit="yearmonth"
  title="Weekly Signups by Source"
/>
```
